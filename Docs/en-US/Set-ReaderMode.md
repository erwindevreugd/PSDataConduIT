# Set-ReaderMode

## SYNOPSIS
Sets a reader to the specified reader mode.

## SYNTAX

```
Set-ReaderMode [[-Server] <String>] [[-Credential] <PSCredential>] -PanelID <Int32> -ReaderID <Int32>
 -Mode <ReaderMode>
```

## DESCRIPTION
Sets a reader to the specified reader mode. 

If the result returns null, try the parameter "-Verbose" to get more details.
