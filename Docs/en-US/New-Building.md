# New-Building

## SYNOPSIS
Creates a new building.

## SYNTAX

```
New-Building [[-Server] <String>] [[-Credential] <PSCredential>] -Name <String> [-SegmentID <Int32>]
 [<CommonParameters>]
```

## DESCRIPTION
Creates a new building.

If the result returns null, try the parameter "-Verbose" to get more details.
