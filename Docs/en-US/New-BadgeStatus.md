# New-BadgeStatus

## SYNOPSIS
Adds a new badge status.

## SYNTAX

```
New-BadgeStatus [[-Server] <String>] [[-Credential] <PSCredential>] -Name <String>
```

## DESCRIPTION
Adds a new badge status to the database. 

If the result return null, try the parameter "-Verbose" to get more details.
