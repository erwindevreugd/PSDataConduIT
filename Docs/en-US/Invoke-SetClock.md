# Invoke-SetClock

## SYNOPSIS
Sets the panel clock.

## SYNTAX

```
Invoke-SetClock [[-Server] <String>] [[-Credential] <PSCredential>] -PanelID <Int32> [-PassThru]
 [<CommonParameters>]
```

## DESCRIPTION
Sets the panel clock.

If the result returns null, try the parameter "-Verbose" to get more details.
