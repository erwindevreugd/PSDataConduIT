# New-Division

## SYNOPSIS
Adds a new division.

## SYNTAX

```
New-Division [[-Server] <String>] [[-Credential] <PSCredential>] -Name <String> [-SegmentID <Int32>]
```

## DESCRIPTION
Adds a new division to the database. 

If the result return null, try the parameter "-Verbose" to get more details.
