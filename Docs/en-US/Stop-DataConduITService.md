# Stop-DataConduITService

## SYNOPSIS
Stops the DataConduIT service.

## SYNTAX

```
Stop-DataConduITService [[-Server] <String>] [-Credential <PSCredential>] [-PassThru]
```

## DESCRIPTION
Stops the DataConduIT service. 

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Stop-DataConduITService
```
