# Invoke-LaunchGuardTour

## SYNOPSIS
Start the guard tour.

## SYNTAX

```
Invoke-LaunchGuardTour [[-Server] <String>] [[-Credential] <PSCredential>] -GuardTourID <Int32>
 -BadgeID <Int64> -MonitoringZoneID <Int32> [-PassThru] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Starts the guard tour.

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### EXAMPLE 1
```
Invoke-LaunchGuardTour
```
