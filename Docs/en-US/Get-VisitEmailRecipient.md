# Get-VisitEmailRecipient

## SYNOPSIS
Gets a visit email recipient.

## SYNTAX

```
Get-VisitEmailRecipient [[-Server] <String>] [[-Credential] <PSCredential>] -VisitID <Int32>
```

## DESCRIPTION
Gets a visit email recipient. 

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-VisitEmailRecipient
```
