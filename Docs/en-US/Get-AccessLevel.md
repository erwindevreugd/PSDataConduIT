# Get-AccessLevel

## SYNOPSIS
Gets an access level.

## SYNTAX

```
Get-AccessLevel [[-Server] <String>] [[-Credential] <PSCredential>] [-AccessLevelID <Int32>] [-Name <String>]
 [-SegmentID <Int32>] [<CommonParameters>]
```

## DESCRIPTION
Gets all access levels or a single access level if an access level id or name is specified.

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### EXAMPLE 1
```
Get-AccessLevel
```
