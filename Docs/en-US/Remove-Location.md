# Remove-Location

## SYNOPSIS
Removes a location.

## SYNTAX

```
Remove-Location [[-Server] <String>] [[-Credential] <PSCredential>] [-LocationID <Int32>] [-SegmentID <Int32>]
 [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes a location.

If the result returns null, try the parameter "-Verbose" to get more details.
