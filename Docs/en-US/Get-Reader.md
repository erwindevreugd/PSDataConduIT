# Get-Reader

## SYNOPSIS
Gets a reader.

## SYNTAX

```
Get-Reader [[-Server] <String>] [[-Credential] <PSCredential>] [-PanelID <Int32>] [-ReaderID <Int32>]
 [-Name <String>] [-HostName <String>] [<CommonParameters>]
```

## DESCRIPTION
Gets all reader or a single reader if a panel id and a reader id are specified.

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### EXAMPLE 1
```
Get-Reader
```
