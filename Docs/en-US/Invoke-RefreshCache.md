# Invoke-RefreshCache

## SYNOPSIS
Refreshes the DataConduIT Manager cache.

## SYNTAX

```
Invoke-RefreshCache [[-Server] <String>] [[-Credential] <PSCredential>]
```

## DESCRIPTION
Refreshes the DataConduIT Manager cache.

If the result return null, try the parameter "-Verbose" to get more details.
