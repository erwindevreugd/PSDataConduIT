# Invoke-RefreshCache

## SYNOPSIS
Refreshes the DataConduIT Manager cache.

## SYNTAX

```
Invoke-RefreshCache [[-Server] <String>] [[-Credential] <PSCredential>] [<CommonParameters>]
```

## DESCRIPTION
Refreshes the DataConduIT Manager cache.

If the result returns null, try the parameter "-Verbose" to get more details.
