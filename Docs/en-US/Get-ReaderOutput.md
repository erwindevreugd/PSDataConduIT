# Get-ReaderOutput

## SYNOPSIS
Gets a reader output.

## SYNTAX

```
Get-ReaderOutput [[-Server] <String>] [[-Credential] <PSCredential>] [-PanelID <Int32>] [-ReaderID <Int32>]
 [-ReaderOutputID <Int32>] [-Name <String>] [<CommonParameters>]
```

## DESCRIPTION
Gets all reader output or a single reader output if a panel id, reader id and reader output id are specified.

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### EXAMPLE 1
```
Get-ReaderOutput
```
