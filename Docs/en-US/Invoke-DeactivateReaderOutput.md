# Invoke-DeactivateReaderOutput

## SYNOPSIS
Deactivates a reader output.

## SYNTAX

```
Invoke-DeactivateReaderOutput [[-Server] <String>] [[-Credential] <PSCredential>] [-PanelID <Int32>]
 [-ReaderID <Int32>] [-ReaderOutputID <Int32>] [-PassThru]
```

## DESCRIPTION
Deactivates a raeder output. 

If the result returns null, try the parameter "-Verbose" to get more details.
