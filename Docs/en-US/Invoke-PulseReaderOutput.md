# Invoke-PulseReaderOutput

## SYNOPSIS
Pulses a reader output.

## SYNTAX

```
Invoke-PulseReaderOutput [[-Server] <String>] [[-Credential] <PSCredential>] [-PanelID <Int32>]
 [-ReaderID <Int32>] [-ReaderOutputID <Int32>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Pulses a raeder output.

If the result returns null, try the parameter "-Verbose" to get more details.
