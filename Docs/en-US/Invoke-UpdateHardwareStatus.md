# Invoke-UpdateHardwareStatus

## SYNOPSIS
Updates the panel hardware status.

## SYNTAX

```
Invoke-UpdateHardwareStatus [[-Server] <String>] [[-Credential] <PSCredential>] -PanelID <Int32> [-PassThru]
```

## DESCRIPTION
Updates the panel hardware status.

If the result returns null, try the parameter "-Verbose" to get more details.
