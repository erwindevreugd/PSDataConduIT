# Remove-User

## SYNOPSIS
Removes an user.

## SYNTAX

```
Remove-User [[-Server] <String>] [[-Credential] <PSCredential>] [-UserID <Int32>] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Removes an user.

If the result returns null, try the parameter "-Verbose" to get more details.
