# Get-BadgeOwner

## SYNOPSIS
Gets a badge owner.

## SYNTAX

```
Get-BadgeOwner [[-Server] <String>] [[-Credential] <PSCredential>] [-BadgeID <Int64>] [-BadgeKey <Int32>]
```

## DESCRIPTION
Gets the owner of a badge. 

If the result returns null, try the parameter "-Verbose" to get more details.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-BadgeOwner
```
